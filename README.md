# 🛒 Amazon Homepage Clone

A frontend clone of the Amazon homepage built using HTML and CSS.  
This project recreates the layout and structure of Amazon’s landing page for learning and practice purposes.

---

## 🚀 Tech Stack

- HTML5
- CSS3
- Flexbox

---

## 📌 Features

- Navigation bar with logo, search bar, and cart section
- Hero banner section
- Product category boxes
- Footer section
- Layout structured similar to Amazon homepage

## 🎯 Project Objective

The purpose of this project was to:

- Practice building real-world webpage layouts
- Improve CSS styling and positioning skills
- Understand e-commerce homepage structure
- Strengthen frontend fundamentals
